HTML Y CSS

El proyecto consta de un documentos HTML, dos documentos CSS y un documentos JavaScript:

Documentos HTML:

  index.html : es la página principal del juego y contiene todos los objetos que lo conforman.
 

Documentos CSS:
En la carpeta CSS se encuentra los archivos empleados:
 	d.css: es la hoja de estilo importada al archivo index.html siempre y cuando el ancho de la pantalla del ordenador supere los 700px.
        s.css: es la hoja de estilo importada al archivo index.html siempre y cuando el ancho de la pantalla del ordenador esté por debajo de los 699px.

Documento JavaScript:
 	js.js En este documento se desarrolla cada una de las funciones que se realizan en el juego.
	

MENU:

Posición Horizontal:

El menú del juego está formado tres botones:Reiniciar, Opciones, Acerca de..

El botón Reiniciar automáticamente reinicia el juego una vez se hace clic sobre él.

El botón Opciones es un menú desplegable, mostrando la opción "Instrucciones" 

El botón Acerca de.. muestra la información sobre el creador del juego.

Posición Vertical:

El menú del juego está formado por el botón "show menu" mostrando las opciones anteriormente explicadas y el botón "hide menú"
